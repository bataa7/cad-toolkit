# CAD工具包 打包优化说明

## 快速开始

直接运行 `build_fast.bat` 脚本，选择打包模式即可。

## 优化措施

### 1. 文件夹模式（推荐）
- **配置文件**: `CAD工具包_文件夹模式.spec`
- **启动速度**: ⭐⭐⭐⭐⭐ 最快
- **优点**: 
  - 无需解压缩，启动速度最快
  - 便于调试和更新
- **缺点**: 
  - 生成多个文件和文件夹
  - 分发时需要打包整个文件夹

### 2. 单文件模式
- **配置文件**: `CAD工具包_优化.spec`
- **启动速度**: ⭐⭐⭐⭐ 较快
- **优点**: 
  - 只有一个exe文件
  - 便于分发
- **缺点**: 
  - 首次启动需要解压到临时目录
  - 启动速度比文件夹模式慢

## 优化技术详解

### 1. 精简依赖
只打包必要的模块：
- pandas（Excel处理）
- openpyxl（Excel读写）
- ezdxf（DXF文件处理）
- PyQt5（界面）

### 2. 排除不需要的库
排除了以下大型库，减少体积和启动时间：
- tkinter
- matplotlib
- scipy
- pytest/unittest
- IPython/jupyter
- setuptools/pip

### 3. 代码优化
- `optimize=2`: 启用Python字节码优化
- 移除调试信息和文档字符串

### 4. 延迟导入
程序代码中已实现延迟导入：
- pandas在需要时才导入
- ezdxf在处理DXF文件时才导入

## 手动打包命令

### 文件夹模式（推荐）
```bash
pyinstaller --clean CAD工具包_文件夹模式.spec
```

### 单文件模式
```bash
pyinstaller --clean CAD工具包_优化.spec
```
