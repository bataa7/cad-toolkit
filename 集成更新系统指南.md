# CAD工具包 - 集成更新系统指南

## 📦 系统组成

### 1. 安装程序
- `installer_config.py` - 安装程序配置生成器
- `installer.iss` - Inno Setup 脚本（自动生成）
- 生成专业的 Windows 安装程序

### 2. 更新管理器
- `update_manager.py` - 内置更新检查和消息推送模块
- 自动检查更新
- 显示更新对话框
- 消息推送功能

### 3. 版本服务器
- `website/version.json` - 版本信息和消息数据
- 支持版本检查
- 支持消息推送

## 🚀 快速集成

### 步骤1：集成更新管理器到主程序

在 `cad_toolkit_gui.py` 中添加：

```python
# 在文件开头导入
from update_manager import integrate_update_manager, CURRENT_VERSION

class MainWindow(QMainWindow):
    def __init__(self):
        super().__init__()
        
        # ... 其他初始化代码 ...
        
        # 集成更新管理器（在界面初始化完成后）
        integrate_update_manager(self)
        
        # 设置窗口标题显示版本号
        self.setWindowTitle(f"CAD工具包 v{CURRENT_VERSION}")
```

就这么简单！更新管理器会自动：
- 添加"帮助"菜单
- 添加"检查更新"和"关于"菜单项
- 启动时自动检查更新（每天一次）
- 显示更新对话框
- 显示消息推送

### 步骤2：创建安装程序

#### 2.1 生成配置文件

```bash
python installer_config.py
```

这会生成：
- `LICENSE.txt` - 许可协议
- `README.txt` - 安装说明
- `installer.iss` - Inno Setup 脚本

#### 2.2 安装 Inno Setup

1. 下载：https://jrsoftware.org/isdl.php
2. 安装 Inno Setup（选择中文语言包）

#### 2.3 编译安装程序

1. 打开 Inno Setup Compiler
2. 文件 -> 打开 -> 选择 `installer.iss`
3. 构建 -> 编译
4. 生成的安装程序在 `installer_output` 目录

## 📋 功能详解

### 自动更新检查

**触发时机：**
- 程序启动时（每天最多一次）
- 用户手动点击"检查更新"

**检查逻辑：**
1. 连接更新服务器
2. 获取最新版本信息
3. 比较版本号
4. 显示更新对话框

**用户选项：**
- 立即下载 - 打开下载页面
- 稍后提醒 - 下次启动再提醒
- 跳过此版本 - 不再提醒此版本

### 消息推送

**消息类型：**
- 欢迎消息
- 功能提示
- 重要通知
- 活动公告

**消息配置：**
```json
{
  "id": "unique_id",
  "title": "标题",
  "icon": "🎉",
  "content": "<p>HTML内容</p>",
  "action_text": "按钮文字",
  "action_url": "链接地址",
  "expire_date": "2026-04-04"
}
```

**显示规则：**
- 每条消息只显示一次
- 过期消息不显示
- 用户可选择"不再显示"

### 强制更新

在 `version.json` 中设置：

```json
{
  "force_update": true,
  "minimum_version": "2.5.0"
}
```

- `force_update: true` - 强制更新提示
- `minimum_version` - 最低支持版本

```python
    update_download_count(version)
    # 返回文件
    return send_file('CAD工具包.exe')
```

## 📊 用户数据

### 存储位置

使用 QSettings 存储用户配置：

```python
settings = QSettings('CADToolkit', 'UpdateManager')
```

**Windows:** `HKEY_CURRENT_USER\Software\CADToolkit\UpdateManager`

### 存储内容

- `auto_check` - 是否自动检查更新
- `last_check` - 上次检查时间
- `skipped_version` - 跳过的版本
- `shown_messages` - 已显示的消息ID列表

### 清除用户数据

```python
settings = QSettings('CADToolkit', 'UpdateManager')
settings.clear()
```

## 🐛 故障排除

### 问题1：无法连接到更新服务器

**解决方案：**
- 检查网络连接
- 检查服务器地址是否正确
- 尝试备用服务器

### 问题2：更新对话框不显示

**解决方案：**
- 检查版本号比较逻辑
- 清除 QSettings 数据
- 检查是否跳过了此版本

### 问题3：消息重复显示

**解决方案：**
- 检查消息ID是否唯一
- 清除 shown_messages 列表
- 检查过期日期设置

### 问题4：安装程序编译失败

**解决方案：**
- 检查 Inno Setup 是否正确安装
- 检查文件路径是否正确
- 查看编译日志
