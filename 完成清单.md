## 📋 集成到主程序

### 自动集成（推荐）

```bash
python integrate_update.py
```

### 手动集成

在 `cad_toolkit_gui.py` 中：

```python
# 1. 添加导入（文件开头）
from update_manager import integrate_update_manager, CURRENT_VERSION

# 2. 在 MainWindow.__init__ 末尾添加
class MainWindow(QMainWindow):
    def __init__(self):
        super().__init__()
        # ... 其他初始化代码 ...
        
        # 集成更新管理器
        integrate_update_manager(self)
        self.setWindowTitle(f'CAD工具包 v{CURRENT_VERSION}')
```

就这么简单！
