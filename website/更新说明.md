当用户访问网站时：
1. 检测localStorage中的已安装版本
2. 与最新版本比较
3. 如有更新，显示通知弹窗
4. 引导用户下载新版本

## 💡 使用方法

### 访问网站

```bash
# 方法1：使用启动脚本
双击 website/启动网站.bat

# 方法2：手动启动
cd website
python -m http.server 8000

# 访问
http://localhost:8000
```

### 发布新版本

```bash
cd website
python update-version.py

# 选择 "1. 发布新版本"
# 输入版本号: 3.1.0
# 选择类型: 2 (功能更新)
# 输入更新内容...
```

### 集成到程序

```python
from version_checker import VersionChecker

CURRENT_VERSION = "3.0.0"
checker = VersionChecker(CURRENT_VERSION)
update_info = checker.check_for_updates()

if update_info and update_info['has_update']:
    show_update_dialog(update_info)
```

## 📝 版本号规范

采用语义化版本：`主版本.次版本.修订号`

- **主版本 (Major)**: 重大更新，可能不兼容
- **次版本 (Minor)**: 功能更新，向下兼容
- **修订号 (Patch)**: Bug修复，向下兼容
