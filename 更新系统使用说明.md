## 🚀 快速开始

### 方式1：演示程序（推荐新手）

1. **启动版本服务器**
   ```bash
   cd website
   python -m http.server 8000
   ```

2. **运行演示程序**
   ```bash
   python demo_update_system.py
   ```

3. **测试功能**
   - 点击"帮助 -> 检查更新"
   - 点击"帮助 -> 关于"
   - 点击"立即检查更新"按钮

### 方式2：集成到主程序

1. **运行集成脚本**
   ```bash
   python integrate_update.py
   ```

2. **手动添加代码**（如果脚本未自动添加）
   
   在 `cad_toolkit_gui.py` 的 `MainWindow.__init__` 末尾添加：
   ```python
   # 集成更新管理器
   integrate_update_manager(self)
   self.setWindowTitle(f'CAD工具包 v{CURRENT_VERSION}')
   ```

3. **测试运行**
   ```bash
   python cad_toolkit_gui.py
   ```

## 🧪 测试更新功能

### 测试1：检查更新（无更新）

**当前状态：** 已是最新版本

1. 确保 `update_manager.py` 中 `CURRENT_VERSION = "3.0.0"`
2. 运行程序
3. 点击"帮助 -> 检查更新"
4. 应显示"当前已是最新版本"

### 测试2：发现新版本

**模拟旧版本：**

1. 修改 `update_manager.py`：
   ```python
   CURRENT_VERSION = "2.5.0"  # 改为旧版本
   ```

2. 运行程序
3. 应自动弹出更新对话框
4. 或点击"帮助 -> 检查更新"

**更新对话框功能：**
- ✓ 显示版本信息
- ✓ 显示更新内容
- ✓ 立即下载按钮
- ✓ 稍后提醒按钮
- ✓ 跳过此版本按钮
- ✓ 自动检查更新选项

### 测试3：消息推送

**查看欢迎消息：**

1. 清除已显示消息记录：
   ```python
   from PyQt5.QtCore import QSettings
   settings = QSettings('CADToolkit', 'UpdateManager')
   settings.remove('shown_messages')
   ```

2. 运行程序
3. 应显示欢迎消息弹窗

**消息功能：**
- ✓ 显示标题和图标
- ✓ HTML格式内容
- ✓ 操作按钮（可选）
- ✓ "不再显示"选项

### 测试4：强制更新

**配置强制更新：**

1. 修改 `website/version.json`：
   ```json
   {
     "force_update": true,
     "minimum_version": "2.5.0"
   }
   ```

2. 使用旧版本运行程序
3. 更新对话框会显示红色警告

## 🐛 常见问题

### Q1: 更新对话框不显示

**原因：**
- 版本号相同或更旧
- 已跳过此版本
- 无法连接服务器

**解决：**
```python
# 清除设置
from PyQt5.QtCore import QSettings
settings = QSettings('CADToolkit', 'UpdateManager')
settings.clear()
```

### Q2: 消息重复显示

**原因：**
- 消息ID重复
- 设置未保存

**解决：**
- 确保每条消息ID唯一
- 检查 QSettings 是否正常工作

### Q3: 无法连接更新服务器

**原因：**
- 服务器未启动
- 网络问题
- URL配置错误

**解决：**
```bash
# 启动本地服务器
cd website
python -m http.server 8000

# 检查URL
curl http://localhost:8000/version.json
```

### Q4: 安装程序编译失败

**原因：**
- 文件路径错误
- Inno Setup 未安装
- 文件缺失

**解决：**
- 检查 installer.iss 中的路径
- 确保所有文件存在
- 查看编译日志
