📥 安装说明

方法1: 使用预编译版本（推荐）

  1. 从Releases页面下载最新的可执行文件
  2. 解压到任意目录
  3. 双击运行 CAD工具包.exe

方法2: 从源码运行

  1. 克隆仓库：git clone https://github.com/bataa7/cad-toolkit.git
  2. 安装依赖：pip install -r requirements.txt
  3. 运行程序：python cad_toolkit_gui.py

系统要求

  • 操作系统：Windows 7/10/11, Linux, macOS
  • Python：3.8 或更高版本
  • 内存：建议 4GB 以上
  • 磁盘空间：至少 500MB
