## 🆘 快速故障排除

### 推送失败？
```bash
# 检查远程仓库
git remote -v

# 重新配置
git remote set-url origin https://github.com/bataa7/cad-toolkit.git
```

### 认证失败？
1. 使用Personal Access Token，不是密码
2. Token需要 `repo` 权限
3. 重新生成: https://github.com/settings/tokens

### 部署失败？
1. 检查 `docs` 目录是否存在
2. 查看Netlify部署日志
3. 确认 `netlify.toml` 配置正确
